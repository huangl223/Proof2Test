Use Proof2Test for test case generation (Windows)

1. In the setting of environment variable "PATH", add the path of the executable "proof2test-v1.exe"; 
2. Go the "EIFGENs/xxx/Proofs" directory, rename the generated boogie program as "autoproof.bpl" and generated model "ce.model";
3. Place the batch file "proof2test.bat" to the same folder, and run the batch file.
